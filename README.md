## 📖  Sobre 

Sistema de biblioteca digital para gerenciamento de acervos, controle de empréstimos/devoluções e consulta ao catálogo de livros. O projeto oferece interfaces distintas para administradores e usuários frequentes.

## 🚀  Funcionalidades

- ✅ **Catálogo digital** de livros
- ✅ **Sistema de empréstimo** com controle de datas
- ✅ **Interface administrativa** para gerenciamento do acervo
- ✅ **Interface para usuários** para consulta e empréstimos
- ✅ **Banco de dados** relacional com SQL Server

## 📋 Pré-requisitos

Antes de começar, certifique-se de ter instalado:

1. **Visual Studio Code** (ou outra IDE)
2. **Git** para versionamento
3. **Node.js** (versão 16 ou superior)
4. **SQL Server Management Studio** (SSMS)
5. **Microsoft SQL Server** (versão 2022 recomendada)

## 🛠️ Instalação

### 1. Clonar o repositório
```bash
git clone https://github.com/EdGomes234/Biblioteca.git
cd Biblioteca
```


### 2. Configurar o banco de dados
```bash
Abra o SQL Server Management Studio

Execute os arquivos SQL na ordem correta (pasta ORDEM_SQL/)

Configure as credenciais no arquivo de configuração (se necessário)
```

### 3. Executar o projeto

#### Interface Administrativa
```bash
cd .\bibliotecaSenai\
node app.js
```

#### Interface de Frequência
```bash
cd .\Formulario-RP\
node app.js
```
